4.1 The organism: Pelagibacter ubique HTCC1062

We are going to work with the genome of Pelagibacter ubique strain HTCC1062 — arguably the most ecologically important bacterium on the planet.

Pelagibacter ubique belongs to the SAR11 clade, a group of free-living marine alphaproteobacteria that may account for 25–50% of all microbial cells in the ocean’s surface waters. Its genome (accession NC_007205.1) is small (~1.3 Mb), streamlined, and extraordinarily well-studied.

Note

Genome streamlining is an evolutionary strategy in nutrient-poor environments: P. ubique has one of the smallest genomes of any free-living organism, with almost no non-coding DNA and very few regulatory elements.

4.2 The FASTA format

A FASTA file is one of the simplest and most widely used formats in bioinformatics. Each sequence entry has two parts:

  1. A header line starting with >, containing the sequence identifier and optional description
  2. One or more lines of sequence data

Example:

>NC_007205.1 Candidatus Pelagibacter ubique HTCC1062, complete sequence
AGTTTTCGAATTTGAATTTTAAGAAGTTTCGAATTTGAATTTGAAGAAATTTCGAATTT
GAATTTGAAAAAGTTTCGAATTTGAATTTGAAGAAATTTCGAATTTGAATTTGAAAAAG
...

FASTA files can contain a single sequence (as here) or many sequences (a multi-FASTA file), which is common for collections of genes or proteins.

4.3 Method 1 — efetch (NCBI Entrez Direct)

If you installed Entrez Direct during setup, you can fetch the genome directly from NCBI.

First, move into your data directory:

cd ~/training/data

Then fetch the genome:

efetch -db nucleotide -id NC_007205.1 -format fasta > pelagibacter_ubique.fasta

Breaking this command down:

Part Meaning
efetch The NCBI fetch tool
-db nucleotide Search the nucleotide database
-id NC_007205.1 The accession number we want
-format fasta Return the sequence in FASTA format
> pelagibacter_ubique.fasta Redirect the output into a file
Note

The > symbol redirects the output of a command into a file instead of printing it to the screen. We will explore redirection more in Chapter 5.

4.4 Method 2 — curl (backup)

If efetch is not available (e.g. on Windows with MobaXterm, or if the installation did not succeed), use curl to download the file directly:

cd ~/training/data
curl -L -o pelagibacter_ubique.fasta \
  https://raw.githubusercontent.com/clarajegousse/training-data/main/pelagibacter_ubique.fasta
Flag Meaning
-L Follow redirects
-o pelagibacter_ubique.fasta Save output to this filename
Tip

wget is an alternative to curl and works the same way for simple downloads:

wget -O pelagibacter_ubique.fasta \
  https://raw.githubusercontent.com/clarajegousse/training-data/main/pelagibacter_ubique.fasta

wget is available on most Linux systems and in MobaXterm. On a Mac, curl is the default.

4.5 Verifying the download

Always check that your file downloaded correctly:

ls -lh pelagibacter_ubique.fasta

You should see a file of around 1.4 MB. If the file is 0 bytes or a few hundred bytes, something went wrong.

Take a quick look at the beginning of the file:

head pelagibacter_ubique.fasta

You should see a header line starting with >NC_007205.1 followed by lines of sequence.

Caution✏️ Exercise 4.1
  1. Download the genome using either efetch or curl (whichever works on your system)
  2. Confirm the file size with ls -lh
  3. Use head to view the first 5 lines
  4. Use tail to view the last 5 lines — what do you notice about the end of the sequence?
NoteSolution 
# Download (choose one method)
efetch -db nucleotide -id NC_007205.1 -format fasta > pelagibacter_ubique.fasta
# or
curl -L -o pelagibacter_ubique.fasta \
  https://raw.githubusercontent.com/clarajegousse/training-data/main/pelagibacter_ubique.fasta

# Check size
ls -lh pelagibacter_ubique.fasta

# Inspect
head -n 5 pelagibacter_ubique.fasta
tail -n 5 pelagibacter_ubique.fasta

The last few lines are just sequence — the FASTA format has no end marker. The genome sequence continues right to the last line of the file.

Important🚀 Bonus — for those who want more

Try fetching the genome in a different format. GenBank format includes annotations (gene positions, product names, etc.):

efetch -db nucleotide -id NC_007205.1 -format gb > pelagibacter_ubique.gb
head -n 30 pelagibacter_ubique.gb

How does this compare to the FASTA file? What additional information does it contain?

Tip🔑 Key points
  • FASTA is a simple two-part format: a > header line followed by sequence data
  • efetch retrieves sequences from NCBI databases directly from the command line
  • curl (or wget) downloads files from URLs — useful as a fallback
  • Always verify downloads with ls -lh and head
